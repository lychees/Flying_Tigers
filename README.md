# 飞虎队：苍穹战棋 / Flying Tigers: Sky Tactics

一款以**二战飞虎队（美国志愿航空队 AVG）**为题材的回合制战棋游戏，玩法参考《曹操传》《机动战士高达》等经典战棋，包含关卡剧情、整备阶段与格纳库养成系统。

## 运行方式

直接用浏览器打开目录下的 `index.html` 即可开始游戏（推荐 Chrome / Edge / Firefox）。

或者启动本地服务器：

```bash
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 游戏特色

- **历史剧情战役**：从缅甸同古训练营到昆明、仰光、马圭、怒江大峡谷，直至衡阳，共 6 个基于史实的关卡。
- **回合制战棋**：在网格地图上移动 P-40 战鹰，利用俯冲、云层、山脉等地形优势击败日军战机。
- **驾驶员成长**：击落敌机获得经验，升级后提升属性并习得技能（神射手、致命一击、空中闪避等）。
- **格纳库整备**：每关之间进入整备阶段，用资金升级机体装甲、引擎、武器，或维修受损战机。
- **多种任务目标**：全灭敌机、保护机场、击落轰炸机群、摧毁地面目标等。

## 历史背景

飞虎队（American Volunteer Group, AVG）是二战期间由陈纳德将军指挥的美国志愿援华航空队，1941 年至 1943 年间在中国、缅甸、印度上空与日军作战。他们以 P-40 战斗机标志性的鲨鱼嘴涂装闻名，在昆明、仰光、怒江等地创造了以少胜多的空战传奇。
